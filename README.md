# Invoice SaaS

A full-stack SaaS application to **create, manage, and track invoices** for clients efficiently. Built with **React** for frontend and **Node.js/Express** for backend.  

---

## 🚀 Features

- **User Authentication** – Secure login and registration.
- **Client Management** – Add, edit, and view clients.
- **Invoice Management** – Create invoices with multiple items.
- **Real-time Calculations** – Automatic total calculation for invoices.
- **Responsive Dashboard** – Overview of invoices, clients, revenue, and pending amounts.
- **Guidance Alerts** – Helps users create clients before creating invoices.
- **Modern UI/UX** – Clean design with attractive colors and interactive elements.
- **Fixed Footer & Scrollable Content** – Professional layout like real SaaS apps.
- **Privacy, Terms & Contact** – Informative footer for users.
